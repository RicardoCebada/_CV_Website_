Maratón de la Ciudad de México, ¿uno de los más exigentes del mundo?: Análisis de Tendencias y Desempeño de los Corredores en las Ediciones 2021-2025

Data Science
Python
En esta entrada, utilizaremos Python para explorar los resultados oficiales de las ediciones 2021 a 2025, con el objetivo de identificar tendencias en el desempeño de los corredores, comparar su rendimiento entre diferentes categorías y ediciones, y contrastar estos resultados con otros maratones internacionales de alto nivel.
Published

September 10, 2025

Descripción del Proyecto

El Maratón de la Ciudad de México es uno de los eventos deportivos más importantes del país, atrayendo a corredores de todo el mundo. Aunque desde el 2018 el maratón CDMX ostenta la prestigiosa categoría Gold Label, no forma parte de los llamados “World Marathon Majors” que incluyen a los maratones de Boston, Londres, Berlín, Chicago, Nueva York y Tokio. Una de las razones principales de esta exclusión es su exigente recorrido que se desarrolla a una altitud promedio de 2,240 metros sobre el nivel del mar, lo que lo convierte en uno de los maratones más difíciles del mundo.

#agregar un bloque de python para mostrar un ejemplo de análisis de datos

import requests
import pandas as pd

def get_data (url, marathon_id):
  payload = {
    "distancia": "MARATON",
    "carreraId": marathon_id, #ID del Maraton
    "categoriaId": ""
}
  try:
    response=requests.post(url, data=payload)
    if response.status_code == 200:
      datos=response.json() #Convertimos en JSON
      df=pd.DataFrame(datos) #Convertimos en DF
      print(f"Datos del ID {marathon_id} obtenidos con éxito")
      return df
    else:
      print(f"Error de esta en el ID {marathon_id}. Error: {response.status_code}")
  except requests.exceptions.RequestException as e:
    print(f"Error en la solicitud POST del ID {marathon_id}: {e}")

#Declaramos un string con la url de MARCATE desde donde se consultan los datos
